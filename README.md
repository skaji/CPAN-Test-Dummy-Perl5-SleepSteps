# NAME

CPAN::Test::Dummy::Perl5::SleepSteps - sleep 10seconds in configure/test steps

# SYNOPSIS

    # will take 10seconds
    > perl Makefile.PL

    # will take 10seconds
    make test

# DESCRIPTION

CPAN::Test::Dummy::Perl5::SleepSteps is for tests of CPAN clients.
